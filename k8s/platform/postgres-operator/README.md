This directory pins the Zalando Postgres Operator bootstrap to chart version `1.14.0`,
which matches the version already running in the Linode cluster.

Use it in two phases on a fresh cluster:

1. `kubectl apply -k k8s/platform/postgres-operator/crds`
2. `kubectl apply -k k8s/platform/postgres-operator/operator`

The rendered manifests are generated by `scripts/render-postgres-operator.sh`.
They are intentionally checked into the repo so cluster recreation does not depend on
manual `helm install` work.
